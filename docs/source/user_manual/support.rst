Contact and Support
===================

For help with the ECG Waveform Annotation App, contact:

Peter Walczyk: ``pwalczyk@med.umich.edu``

For questions regarding clinical content or High-Level project approaches, please contact the PI:

Dr. Hamid Ghanbari: ``ghhamid@med.umich.edu``

When Emailing for Support
-------------------------

Include the following information:

- Subject line: ``ECG Annotation App Support``
- Your U-M uniqname.
- Subject or waveform record UUID if subject specific.
- Base folder path, if relevant.
- Description of the issue.
- Exact error message, if shown.
- Screenshot, if available.
- What you clicked immediately before the issue occurred.

Helpful Details to Include
--------------------------

If the issue involves loading data, include:

- Whether you are on the Michigan Network or VPN.
- Whether the base folder loads successfully.
- Whether waveform records appear in the dropdown.
- Which waveform record you selected.
- Whether the issue happens for one record or multiple records.

If the issue involves annotations, include:

- Your username entered in the app.
- Whether you clicked **Load Subject** before **Load Annotations**.
- Whether the annotation file should be partial or complete.
- Whether you recently used **Remove Last Mark**.
- Whether the waveform had already been finalized.

Do Not Send
-----------

Unless specifically requested and approved, do not email:

- Raw waveform files.
- Patient identifiers. (UUID is okay)
- Protected health information. 
- Full annotation datasets.

Use approved storage and sharing workflows for project data. 
DropBox is the only approved cloud storage platform for PHI. Turbo Storage Volume is our primary storage.
