User Manual
===========

This section is for annotators using the ECG Waveform Annotation App.

Start with **Getting Started** if this is your first time using the app. Use **Walkthrough Videos** for visual step-by-step guidance.

Quick Start
-----------

1. Launch the application.
2. Enter/select your U-M uniqname.
3. Select the base data folder.
4. Choose a waveform record.
5. Click **Load Subject**.
6. Click on the waveform to place an interval endpoint.
7. Answer the CPR and rhythm questions.
8. Click **Mark**.
9. Save periodically with **Save All Annotations**.
10. When the final interval reaches the end of the waveform, click **Finalize Waveform**.
