Frequently Asked Questions
==========================

Can I undo or delete past marks?
--------------------------------

Only the most recent annotation can be removed using **Remove Last Mark**.

Older annotations cannot be directly edited or deleted. This preserves sequential annotation traceability.

Can I edit previous annotations?
--------------------------------

No. Annotations are sequential. To revise an earlier interval, you must remove later intervals first.

Where are annotation files stored?
----------------------------------

Annotation files are stored in an ``output`` folder associated with the waveform record and user.

See :ref:`saving-resuming`.

Can I annotate the same waveform in multiple sessions?
------------------------------------------------------

Yes. Save your work, close the app, and later resume by selecting the same username and waveform record. Then click **Load Subject** and **Load Annotations**.

Why do I need to enter my User Name?
------------------------------------

Your username identifies your annotation file and allows you to resume your own work. Use your U-M uniqname in lowercase.

I do not see my previous annotations. What happened?
----------------------------------------------------

Check that:

- Your username is correct.
- You selected the same waveform record.
- You clicked **Load Subject** before **Load Annotations**.
- The annotation file was not moved, renamed, or deleted.

How often should I save?
------------------------

Autosave runs every 2 minutes.

You should also manually save with **Save All Annotations**, especially before closing the app or taking a break.

What do red sidebar warnings mean?
----------------------------------

Red warnings indicate that a required step is missing or invalid.

Examples include:

- Missing username.
- Missing CPR answer.
- Missing rhythm label.
- Missing explanation.
- Invalid interval.
- Interval shorter than 1 second.

What does the annotation table show?
------------------------------------

The annotation table shows saved intervals, including:

- User.
- Subject.
- CPR response.
- Rhythm label.
- Explanation.
- Start time.
- End time.

What are the colored waveform overlays?
---------------------------------------

Colored overlays show annotated intervals on the waveform.

They provide visual feedback for the segments you have already marked.

What are the vertical dashed lines?
-----------------------------------

Vertical dashed lines show event or recording markers.

Common colors:

- Green: clinical events.
- Red: code start/end.
- Purple/Yellow: recording start/end.

Can I annotate if some leads have missing data?
-----------------------------------------------

Yes. Leads with no data can be ignored. Annotate using available waveform leads.

How do I adjust waveform amplitude?
-----------------------------------

Use the Y-axis controls next to each waveform:

- **Y-IN** to zoom in vertically.
- **Y-OUT** to zoom out vertically.
- **Y+** to shift up.
- **Y-** to shift down.
- **AUTO ON/OFF** to control automatic Y-scaling.

What happens if my interval is too short?
-----------------------------------------

Intervals must be at least 1 second long.

If the interval is too short, the app displays a warning and disables **Mark**.

Why is the Mark button disabled?
--------------------------------

The **Mark** button is disabled until all requirements are met.

See :ref:`troubleshooting`.

Can I manually edit annotation files?
-------------------------------------

No. Do not manually edit, rename, move, or delete annotation files.

Use the app to save, load, and remove marks.

What if the app crashes?
------------------------

Relaunch the app, select the same username and waveform record, click **Load Subject**, then click **Load Annotations**.

Autosave should preserve most recent progress.

Do I need VPN?
--------------

If you are off-site, VPN may be required to access network data.

For best performance, use the Michigan Network directly when possible.

Can I annotate from home?
-------------------------

Yes, if you have the required network or VPN access. Loading may be slower.

Is my annotation data shared with others?
-----------------------------------------

Your annotation files are user-specific. They may later be reviewed or used during adjudication depending on the project workflow.
